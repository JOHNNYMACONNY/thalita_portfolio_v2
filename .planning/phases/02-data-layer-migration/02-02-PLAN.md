## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Local markdown/assets → migration script | Untrusted legacy content is converted into importable CMS documents |
| Migration script → Sanity dataset import | Bulk content write crosses into the canonical CMS dataset |
| Editorial checkpoint → import execution | Human approval determines whether ambiguous mapping is safe to publish |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

</threat_model>
